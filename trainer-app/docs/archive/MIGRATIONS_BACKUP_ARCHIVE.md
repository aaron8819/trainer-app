# Prisma Migration Backup Archive

This folder stores historical Prisma migration snapshots moved out of the active Prisma tree during PR4 of `ARCHITECTURE_CLEANUP_PLAN.md`.

- Active runtime migrations: `prisma/migrations/`
- Archived snapshots: `docs/archive/prisma-migrations-backup/`

Notes:
- Archived files are for historical reference only.
- Prisma migration commands should target only `prisma/migrations/`.
